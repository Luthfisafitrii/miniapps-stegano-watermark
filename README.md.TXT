Mini Apps — Steganografi & Watermark

Aplikasi ini memungkinkan pengguna untuk:
- Menyisipkan pesan rahasia ke gambar menggunakan metode LSB Steganografi.
- Menambahkan watermark ke gambar dengan teks, ukuran font, dan warna yang dapat dipilih.

Cara Menjalankan
1. Pastikan Python sudah terinstall.  
2. Install library
3. Jalankan aplikasi

Struktur Folder
miniapps-stegano-watermark/
│
├── miniapps.py
├── requirements.txt
└── README.md

Library yang Dibutuhkan
- streamlit
- pillow
- numpy
